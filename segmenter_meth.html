These methods are used to handle data and run the segmentation process.

set_target(self, t[, copy]) Sets the target data by overiding any existing target.
add_reference(self, r[, copy]) Appends a reference containing one or more features to the existing reference dataset.
clear_reference(self) Removes any reference data currently assigned
segment(self) Method to run the segmentation algorithm on the current Segmenter instance
